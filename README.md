# Project Nova Mobile

Phone-first fork of Project Nova for Android, built with Godot 4.7.

## Touch controls

- Left stick: movement
- Right stick: aim and automatic fire
- RYV: dash
- NOVA: companion pulse
- E: interact
- ORUZH: switch weapon
- GRAN / SHCHIT: active items
- MAP: floor map
- Gear icon: move every control and change the global control size

All keyboard and gamepad mechanics remain available for desktop testing.

## APK

Every push to `main` runs **Build Android APK**. Open the latest successful workflow run and download the `ProjectNova-Mobile-APK` artifact.
